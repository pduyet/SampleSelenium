# Pre-Requisites:
    1. Install JDK 11 (version 11.0.14)
    2. Install IDE (preferred IntelliJ IDEA)
    3. Install Maven
    4. Install Allure


## Clone the project
  `git clone https://gitlab.com/setel/automation/setel-automated-test.git`

### Run sample testcase

  `mvn -Dtest=Authorization_Login test`
